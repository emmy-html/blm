Protect Black Lives was created by Emmy Jordan as a resource to educate and spread awareness regarding the police brutality crisis that affects Black lives worldwide. This website will provide racism on how to combat and educate yourself on the consequences of systemic racism, and police brutality.

How to Contribute

This project is super beginner-friendly. If you would like to contribute to this project, check out the repo and read the README to find out how you can update the collective.


Don't know how to code? No worries! You can submit your resource using this form. Please give at least 24 hours for your submission to be reviewed and added to the collective.
